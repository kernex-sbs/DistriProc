  <action>Create a C program that:
1. Allocates anonymous memory with mmap (MAP_PRIVATE | MAP_ANONYMOUS)
2. Registers it with userfaultfd (UFFDIO_REGISTER_MODE_MISSING)
3. Spawns a background thread to handle faults
4. In the main thread, writes to the uninitialized memory (triggering faults)
5. In the handler thread, reads uffd_msg, allocates a page, zero-fills it, and installs it via UFFDIO_COPY
6. Verifies the write succeeded after the fault is handled
